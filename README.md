# 项目简介
这是用vue写的一个模仿最新的苹果官网的项目，截止时间为2020年7月2日。
项目完成了苹果官网首页的还原，并对个别产品进行了详细的介绍。
没有使用任何第三方组件库，组件全部都是自己纯手写。
其中还原了airpods pro产品的详细介绍，具有极强的交互式体验，用了大量的js原生操作dom的方法。
响应式布局，兼容移动端和pc端。
技术栈：vue-cli、vue、vue-router
静态资源(图片)来源:苹果中国官网
